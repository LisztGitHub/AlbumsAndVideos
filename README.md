## 视频，提供基础，UI控制层可以自定义写，视频包含各种手势操作

>连击两下 暂停播放
>
>两个指头 全屏或返回原样
>
>水平左右滑动 快进快退
>
>垂直左右两边上下滑动 调整亮度和音量
